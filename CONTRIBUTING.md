Getting Started
================
-> Fork the repo and clone your fork 
-> Create a new branch for each feature or fix
-> Commit your work and push to your fork after editing
-> Open a pull request to merge your branch into the team repo
-> Tag the Lead [@dnichols28] for review



⭐Commit Message Rules⭐
=========================
-> Credit is based only on commit history
-> Your name MUST be on the commit to recieve credit
-> Keep messages short and clear

⭐Issue and Workflow Process⭐
===============================
-> All work must start with a GitHub issue  
-> No one makes changes unless it’s connected to an issue  
-> The Lead assigns issues using @githubhandle  
-> Each member must:  
  - Complete 5 assigned issues  
  - Complete 5 unassigned issues  
  - Create 5 new unassigned issues  
-> Use comments to update progress on your assigned issues

⭐Testing Requirements⭐
=========================
-> Use a testing framework for all tests  
  -Java: JUnit 5  
  -Python: unittest  

Breakdown:
- Centennial Campaign Impact function → 10 tests (SWE and Tester)  
- SWE UML class → 10 tests (SWE)  
- Lead/Designer UML class → 20 tests (Lead and Designer)  
- Run all tests before creating a PR

  

⭐Coding Standards⭐
=====================
- Follow repo’s python language style
- Keep code clean, readable, and commented  
- Follow UML diagram exactly unless new diagrams are uploaded

  

 ⭐Project Organization⭐
==========================
/Assets - Brand images, favicon, UML diagrams | /src - Source code | /tests - Unit tests | /drivers - XULA driver  | /docs - Documentation (optional)  


⚠ Pull Request Process ⚠
===========================
1] Push your branch (git push origin feature/your-feature-name)  
2] Open a pull request on GitHub  
3] Write a short description of what you did and what issue it fixes  
4] Wait for Lead approval before merging



⭐Communication⭐
==================
- Use GitHub Issues to plan and track all work  
- Mention teammates with @githubhandle to coordinate  
- Keep communication respectful and clear
